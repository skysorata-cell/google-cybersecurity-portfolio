# Botium Toys – Internal Security Audit

## Overview
This project documents an internal security audit conducted for **Botium Toys**, a fictional U.S.-based retail company with a growing online presence.  
The purpose of this audit is to assess the organization’s current security posture, identify risks to critical assets, and recommend improvements aligned with industry standards and regulatory requirements.

The audit is based on the **NIST Cybersecurity Framework (CSF)** and evaluates security controls, compliance obligations, and risk management practices.

---

## Audit Objectives
- Identify risks, threats, and vulnerabilities to organizational assets
- Evaluate existing security controls
- Assess compliance with relevant regulations
- Provide actionable recommendations to improve security posture

---

## Scope
The audit covers the entire security program at Botium Toys, including:
- Employee devices and IT-managed assets
- Internal network and internet access
- On-premises and legacy systems
- Data storage, retention, and processing
- Physical facilities (office, storefront, warehouse)
